1. JWT概念

​ Json web token, 是为了在网络应用环境中传递声明而执行的一种基于json的开放标准。特别适合分布式站点的单点登录场景。JWT的声明一般被用来在身份提供者和服务提供者之间传递被认证的用户身份信息，以便于从服务器获取资源，也可以增加一些额外的其他业务逻辑所用到的生命信息。

​ JWT是由3段信息构成的，将这三段信息用.链接到一起就成了JWT字符串。

​ JWT第一部分称之为头部，第二部分称之为载荷，第三部分是签证。

1.1 Header

头部承担两部分信息：

  • 声明类型 jwt

  • 加密算法 HMAC SHA256等

    完整的头部如下，将头部进行base64加密，构成了第一部分：

{
'typ': 'JWT',
'alg': 'HS256'
}

1.2 Playload

载荷就是存放有效信息的地方，包含三部分

  • 标准中注册的声明 (建议但不强制使用)

  • 公共的声明：

    ​ 公共的声明可以添加任何的信息，一般添加用户的相关信息或其他业务需要的必要信息.但不建议 添加敏感信息，因为该部分在客户端可解密

  • 私有的声明：

    ​ 私有声明是提供者和消费者所共同定义的声明，一般不建议存放敏感信息，因为base64是对称解 密的，意味着该部分信息可以归类为明文信息

    参数 说明
    iss jwt签发者
    sub jwt所面向的用户
    aud 接收jwt的一方
    exp jwt的过期时间，这个过期时间必须要大于签发时间
    nbf 定义在什么时间之前，该jwt都是不可用的.
    iat jwt的签发时间
    jti jwt的唯一身份标识，主要用来作为一次性token,从而回避重放攻击。

1.3 Signature

​ jwt的第三部分是一个签证信息，这个签证信息由三部分组成：

  • Header (base64后的)

  • Payload (base64后的)

  • Secret

    ​ 这个部分需要base64加密后的header和base64加密后的payload使用.连接组成的字符串， 然后通过header中声明的加密方式进行加盐secret组合加密，然后就构成了jwt的第三部分。

2. JWT的生成代码（Java）


private static String generateToken(Map<String, Object> claims, long durationMs){
        logger.info("[generateToken]|claims:{}", JSONObject.toJSON(claims));
        byte[] apiKeySecretBytes = DatatypeConverter.parseBase64Binary(JWT_SECRET);
        PKCS8EncodedKeySpec signingKey = new PKCS8EncodedKeySpec(apiKeySecretBytes);
        PrivateKey privateKey = null;
        try {
            privateKey = KeyFactory.getInstance("RSA").generatePrivate(signingKey);
        } catch (InvalidKeySpecException | NoSuchAlgorithmException e ) {
            e.printStackTrace();
        }
        Instant expireDate = Instant.now().plusMillis(durationMs);
        return Jwts.builder()
                .setClaims(claims)
                .setIssuedAt(Date.from(Instant.now()))
                .setExpiration(Date.from(expireDate))
                .signWith(SignatureAlgorithm.RS256, privateKey)
                .compact();
    }

    private static String generateRefreshToken(String accessToken){
        RefreshToken refreshToken = RefreshToken.builder()
                .expire(Instant.now().plusMillis(JWT_REFRESH_TOKEN_DURATION * 1000))
                .token(accessToken)
                .refreshCount(REFRESH_DEFAULT_COUNT)
                .build();
        Map<String, Object> claims = new HashMap<>();
        claims.put(REFRESH_TOKEN, refreshToken);
        return generateToken(claims, JWT_REFRESH_TOKEN_DURATION * 1000);
    }

    private static String generateAccessToken(String username){
        Map<String, Object> claims = new HashMap<>();
        claims.put(USERNAME, username);
        return generateToken(claims, JWT_ACCESS_TOKEN_DURATION * 1000);
    }

//校验JWT 
public static StatusResult validateToken(String authToken) {
        StatusResult statusResult = StatusResult.success();
        try {
            parseToken(authToken,JWT_PUBLIC_SECRET);
        } catch (SignatureException ex) {
            statusResult.update(AbstractConstant.SSOStatus.LOGIN_ERROR_TOKEN, "Invalid JWT signature");
        } catch (MalformedJwtException ex) {
            statusResult.update(AbstractConstant.SSOStatus.LOGIN_ERROR_TOKEN, "Invalid JWT token");
        } catch (ExpiredJwtException ex) {
            statusResult.update(AbstractConstant.SSOStatus.LOGIN_ERROR_TOKEN, "Expired JWT token");
        } catch (UnsupportedJwtException ex) {
            statusResult.update(AbstractConstant.SSOStatus.LOGIN_ERROR_TOKEN, "Unsupported JWT token");
        } catch (IllegalArgumentException ex) {
            statusResult.update(AbstractConstant.SSOStatus.LOGIN_ERROR_TOKEN, "JWT claims string is empty.");
        } catch (NoSuchAlgorithmException e) {
            statusResult.update(AbstractConstant.SSOStatus.LOGIN_ERROR_TOKEN, "Invalid JWT Algorithm");
        } catch (InvalidKeySpecException e) {
            statusResult.update(AbstractConstant.SSOStatus.LOGIN_ERROR_TOKEN, "Invalid JWT KeySpec");
        } finally {
            return statusResult;
        }
    }

//解析JWT
    public static Jws<Claims> parseToken(String authToken, String jwtPubSecret)
            throws NoSuchAlgorithmException, InvalidKeySpecException,SignatureException {
        X509EncodedKeySpec keySpecX509 = new X509EncodedKeySpec(Base64.getDecoder().decode(jwtPubSecret));
        RSAPublicKey pubKey = (RSAPublicKey) KeyFactory.getInstance("RSA").generatePublic(keySpecX509);
        return Jwts.parser().setSigningKey(pubKey).parseClaimsJws(authToken);
    }
